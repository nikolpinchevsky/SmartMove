## Prototype Description

The SmartMove prototype is a mobile application designed to manage moving boxes during a home relocation.

Each box is represented digitally in the system and includes information such as name, destination room, priority level, fragile/valuable indicators, and status.

A unique QR code is generated for each box. By scanning the QR code, the user can instantly access the box details without opening it.

The prototype supports:
- creating and managing moving projects
- creating boxes with attributes
- searching and filtering boxes
- updating box status
- displaying a priority list of boxes to open first

An optional AI-based feature for analyzing box contents exists but was not fully tested in this prototype.

The prototype demonstrates how SmartMove improves organization and control during the moving process.
